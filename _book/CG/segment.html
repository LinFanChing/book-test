线段

直线与线段的表示方法

我们可以用一条线段的两个端点来表示一条线段。直线的表示有两种方式，一种方式是使用二元一次方程y=kx+by=kx+by=kx+b来表示，另一种是用直线上任意一条长度不为零的线段来表示。由于使用方程表示接近垂直于某坐标轴的直线时容易产生精度误差，所以我们通常使用直线上的某条线段来表示直线。

struct Sg {
    Pt s, t;
    Sg() { }
    Sg(Pt s, Pt t) : s(s), t(t) { }
    Sg(double a, double b, double c, double d) : s(a, b), t(c, d) { }
};

点在线段上的判断

判断点CCC在线段ABABAB上的两条依据：

  1. CA⃗⋅CB⃗=0\vec{CA}\cdot\vec{CB} = 0​CA​⃗​​⋅​CB​⃗​​=0。
  2. CCC在以ABABAB为对角顶点的矩形内。

示例代码

bool PtOnSegment(Pt s, Pt t, Pt a) {
    return !det(a-s, a-t) && min(s.x, t.x) <= a.x && a.x <= max(s.x, t.x) &&
        min(s.y, t.y) <= a.y && a.y <= max(s.y, t.y);
}

另一种方法

判断点CCC在ABABAB为对角线定点的矩形内较麻烦，可以直接判断CA⃗⋅CB⃗\vec{CA}\cdot\vec{CB}​CA​⃗​​⋅​CB​⃗​​的符号来判断CCC在直线ABABAB上是否在ABABAB之间。

示例代码

bool PtOnSegment(Pt p, Pt a, Pt b) {
    return !sgn(det(p-a, b-a)) && sgn(dot(p-a, p-b)) <= 0;
}

把上例代码中的<=改成==就能实现不含线段端点的点在线段上的判断。

点在直线上的判断

点在直线上的判断很简单只要把点在线段上的判断的步骤2去掉即可。

示例代码

bool PtOnLine(Pt p, Pt s, Pt t) {
    return !sgn(det(p-a, b-a));
}

求点到直线的投影

示例代码

Pt PtLineProj(Pt s, Pt t, Pt p) {
    double r = dot(p-s, t-s) / (t - s).norm();
    return s + (t - s) * r;
}

判断直线关系

直线有相交和平行两种关系，靠叉乘能简单判断。

bool parallel(Pt a, Pt b, Pt s, Pt t) {
    return !sgn(det(a-b, s-t));
}

判断线段关系

线段有相交和不相交两种关系，通常按照以下步骤判断。

  1. 快速排斥试验
  2. 跨立试验

快速排斥试验

设以线段P1P2P_1P_2P​1​​P​2​​为对角线的矩形为RRR，设以线段Q1Q2Q_1Q_2Q​1​​Q​2​​为对角线的矩形为TTT，如果RRR和TTT不相交，显然两线段不会相交。

跨立试验

如果两线段相交，则两线段必然相互跨立对方。若P1P2P_1P_2P​1​​P​2​​跨立Q1Q2Q_1Q_2Q​1​​Q​2​​，则矢量Q1P1⃗\vec{Q_1P_1}​Q​1​​P​1​​​⃗​​和Q1P2⃗\vec{Q_1P_2}​Q​1​​P​2​​​⃗​​位于矢量Q1Q2⃗\vec{Q_1Q_2}​Q​1​​Q​2​​​⃗​​的两侧，即Q1P1⃗×Q1Q2⃗⋅Q1Q2⃗×Q1Q2⃗<0\vec{Q_1P_1} \times \vec{Q_1Q_2} \cdot \vec{Q_1Q_2} \times \vec{Q_1Q_2} < 0​Q​1​​P​1​​​⃗​​×​Q​1​​Q​2​​​⃗​​⋅​Q​1​​Q​2​​​⃗​​×​Q​1​​Q​2​​​⃗​​<0。上式可改写成Q1P1⃗×Q1Q2⃗⋅Q1Q2⃗×Q1P2⃗>0\vec{Q_1P_1} \times \vec{Q_1Q_2} \cdot \vec{Q_1Q_2} \times \vec{Q_1P_2} > 0​Q​1​​P​1​​​⃗​​×​Q​1​​Q​2​​​⃗​​⋅​Q​1​​Q​2​​​⃗​​×​Q​1​​P​2​​​⃗​​>0。当Q1P1⃗×Q1Q2⃗=0\vec{Q_1P_1} \times \vec{Q_1Q_2} = 0​Q​1​​P​1​​​⃗​​×​Q​1​​Q​2​​​⃗​​=0时，说明Q1P1⃗\vec{Q_1P_1}​Q​1​​P​1​​​⃗​​和Q1Q2⃗\vec{Q_1Q_2}​Q​1​​Q​2​​​⃗​​共线，但是因为已经通过快速排斥试验，所以P1P_1P​1​​一定在线段Q1Q2Q_1Q_2Q​1​​Q​2​​上；同理，Q1Q2⃗×Q1P2⃗=0\vec{Q_1Q_2} \times \vec{Q_1P_2} = 0​Q​1​​Q​2​​​⃗​​×​Q​1​​P​2​​​⃗​​=0说明P2P_2P​2​​一定在线段Q1Q2Q_1Q_2Q​1​​Q​2​​上。所以判断P1P2P_1P_2P​1​​P​2​​跨立Q1Q2Q_1Q_2Q​1​​Q​2​​的依据是：Q1P1⃗×Q1Q2⃗⋅Q1Q2⃗ timesvecQ1P2≥0\vec{Q_1P_1} \times \vec{Q_1Q_2} \cdot \vec{Q_1Q_2} \ times vec{Q_1P_2} \geq 0​Q​1​​P​1​​​⃗​​×​Q​1​​Q​2​​​⃗​​⋅​Q​1​​Q​2​​​⃗​​ timesvecQ​1​​P​2​​≥0。同理判断Q1Q2Q_1Q_2Q​1​​Q​2​​跨立P1P2P_1P_2P​1​​P​2​​的依据是：P1Q1⃗×P1P2⃗⋅P1P2⃗×P1Q2⃗≥0\vec{P_1Q_1} \times \vec{P_1P_2} \cdot \vec{P_1P_2} \times \vec{P_1Q_2} \geq 0​P​1​​Q​1​​​⃗​​×​P​1​​P​2​​​⃗​​⋅​P​1​​P​2​​​⃗​​×​P​1​​Q​2​​​⃗​​≥0。

求点到线段的距离

求线段ababab到点p最短距离的方法为：

根据点ppp到的投影点的位置进行判断的方法：

  1. 判断线段papapa和ababab所成的夹角，如果是钝角，那么∣pa∣|pa|∣pa∣是点到线段的最短距离。
  2. 判断线段pbpbpb和ababab所成的夹角，如果是钝角，那么∣pb∣|pb|∣pb∣是点到线段的最短距离。
  3. 线段papapa和线段pbpbpb与ababab所成的夹角都不为钝角，那么点ppp到线段ababab的距离是点ppp到直线ababab的距离，这个距离可以用面积法直接算出来。

示例代码

double PtSegmentDist(Pt a, Pt b, Pt p) {
    if (sgn(dot(p-a, b-a)) <= 0) return (p-a).norm();
    if (sgn(dot(p-b, a-b)) <= 0) return (p-b).norm();
    return fabs(det(a-p, b-p)) / (a-b).norm();
}
